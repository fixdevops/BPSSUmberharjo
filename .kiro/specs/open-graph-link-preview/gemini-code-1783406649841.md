# Dokumentasi Keuangan: Peternakan

## 1. Komponen Utama Input Data
Setiap transaksi peternakan harus menyertakan variabel berikut:
- **Populasi:** Jumlah ekor ternak.
- **Biaya Pakan/Produksi:** Total biaya variabel pakan/vaksin.
- **Biaya Operasional:** Biaya pendukung (listrik, air, dll).
- **Tenaga Kerja:** - `jumlah_pekerja` (int)
    - `upah_per_pekerja` (decimal)
- **Nilai Produksi:** Estimasi hasil penjualan total.

## 3. Logika Perhitungan untuk Developer
Sistem harus mengeksekusi logika ini untuk memastikan akurasi (terutama pada kondisi pekerja mandiri):

```javascript
// Algoritma Penentuan Gaji
const hitungTotalGaji = (jumlahPekerja, upahPerPekerja) => {
  if (jumlahPekerja <= 0 || upahPerPekerja <= 0) {
    return 0; // Dikelola pemilik
  }
  return jumlahPekerja * upahPerPekerja;
};
